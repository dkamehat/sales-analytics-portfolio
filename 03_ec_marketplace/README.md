# Phase 3 — EC Marketplace 🚧

**Status:** Planned, not yet built.

Third application of the framework to a fictional EC marketplace seller-acquisition business. Tests whether the framework holds for a **transactional high-volume business** with very different unit economics than B2B SaaS.

## Planned scope

- Sales Performance Overview (seller signup → first listing → first sale conversion)
- Pipeline Health (seller activation funnel)
- Account Insights (seller cohort GMV trajectory, category × segment ARPA)
- Seller Health (Phase 1's CS dashboard, recast as "seller success")

## What this tests

In B2B SaaS, an account is acquired once and monetized over years. In EC marketplaces, sellers can churn between listings, scale dramatically when a SKU goes viral, and re-activate without sales involvement. The framework needs to handle high-frequency state changes — that's the test.
