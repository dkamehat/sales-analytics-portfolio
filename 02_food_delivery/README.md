# Phase 2 — Food Delivery 🚧

**Status:** Planned, not yet built.

This phase will apply the same four-dashboard framework from Phase 1 to a synthetic food-delivery merchant-acquisition business, then extend with an urban-vs-regional geography view.

## Planned scope

### Same as Phase 1
- Sales Performance Overview (merchant commission revenue)
- Pipeline Health (onboarding velocity by region)
- Account Insights (cuisine × segment compounding)

### Food-delivery-specific additions
- **Active merchant cohort retention** — what % of merchants acquired in month N are still active 6 months later
- **GMV trajectory by cohort** — leading vs lagging cohorts
- **Visit-vs-Call decision matrix** — input to the Phase 5 field-sales optimization work

## Why this domain matters

The framework from Phase 1 transfers cleanly here, and Phase 5 (field sales optimization) builds directly on the data shape introduced here.
